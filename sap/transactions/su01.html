SAP SU01

SU01 is used to maintain and create a user account.

What is SU01 in SAP?

SU01 is the transaction code for User Maintenance in SAP. It is used by administrators to manage user accounts in the system, including creating, editing, locking, or unlocking users. This transaction is essential for managing user access rights and ensuring that users have the correct authorizations to perform their roles.

Common Tasks in SU01:

  • Create a new user: Add a new user to the SAP system.
  • Change user details: Update user information such as roles, profiles, and authorization.
  • Lock/Unlock users: Temporarily disable or enable a user account.
  • Assign roles and profiles: Assign specific roles and permissions to a user to control what they can access in SAP.

SU01 is crucial for user management and security, as it helps control which users have access to various functions and data within the SAP system.
